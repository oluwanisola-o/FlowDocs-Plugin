Where to get icons & illustrations

This file uses inline SVGs (Feather-style). For production, use assets that match Alama/Lifesum’s friendly, minimal look:

  • Icons — Feather Icons, Heroicons, Material Symbols: thin strokes, 24×24, work well on dark UI.
  • Offline / error illustrations — unDraw (editable color), Blush, Freepik: simple scenes (e.g. “no connection”, “something went wrong”). Pick one style and stick to it.
  • Lifesum/Alama-style — Rounded, soft shapes and limited palette (e.g. your green + neutral). Use a single illustration set (e.g. from Blush or unDraw) for all error/empty states so the app feels consistent.
